Làm chủ Java Stream API: Xử lý Collections chưa bao giờ dễ đến thế

20 tháng 12, 2024

Giới thiệu

Trước khi có Java 8, việc thao tác với Collections (ví dụ: lọc danh sách, chuyển đổi kiểu dữ liệu, tính tổng) thường đòi hỏi nhiều vòng lặp for hoặc while lồng nhau, làm cho code trở nên dài dòng và khó bảo trì. Stream API ra đời như một cuộc cách mạng, giúp chúng ta viết code theo phong cách lập trình hàm (functional programming) ngắn gọn và rõ ràng hơn.

Ví dụ thực tế: Quản lý sinh viên

Giả sử chúng ta có một danh sách sinh viên và muốn thực hiện các yêu cầu sau:

  1. Lọc ra các sinh viên có điểm trung bình (GPA) >= 8.0.
  2. Lấy danh sách tên của các sinh viên đó.
  3. Sắp xếp tên theo thứ tự bảng chữ cái.

Cách truyền thống (Dùng vòng lặp)

List<Student> students = //... danh sách sinh viên
List<String> goodStudentNames = new ArrayList<>();

for (Student s : students) {
    if (s.getGpa() >= 8.0) {
        goodStudentNames.add(s.getName());
    }
}
Collections.sort(goodStudentNames);
